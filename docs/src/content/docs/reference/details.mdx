</details>

## Steps

## In steps

The following is a `<details>` element nested in a `<Steps>` component:

import { Steps } from '@astrojs/starlight/components';

<Steps>

1.  Step one has some information

      <details>

        <summary>Need to know more?</summary>

        You might need more information for step one.

        So we're here to help.

      </details>

2.  Step two has some more information

</Steps>

## With steps

The following is a `<details>` element containing a `<Steps>` component:

<details>
  <summary>Need more details?</summary>
  <Steps>

1. Step one has some information.

   These steps help if you need extra details to complete this task.

2. Step two has some more information.

  </Steps>
</details>
